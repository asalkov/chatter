# Components Directory

This directory will contain all React components organized by feature:

## Planned Structure

- `auth/` - Authentication components (Login, Register, etc.)
- `chat/` - Chat-related components (MessageList, MessageInput, etc.)
- `layout/` - Layout components (Sidebar, Header, etc.)
- `common/` - Reusable components (Button, Modal, etc.)

Components will be added as development progresses.
